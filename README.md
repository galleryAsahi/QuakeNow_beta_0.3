# QuakeNow_beta_0.2
地震に関する情報を表示するソフトです。まだベータ版であり、動作は保証されないことに留意してください。

# 各種情報の引用元  
強震モニタ：[国立研究開発法人 防災科学技術研究所の長周期地震動モニタ](https://www.lmoni.bosai.go.jp/monitor/)からの引用です。*1  
海底地震計：[海上保安庁の海しる](https://www.msil.go.jp/portal/apps/sites/#/msil-o-top)からの引用です。*1  

**注釈**  
*1 [リアルタイム地震ビューアー TurboWarp版](https://turbowarp.org/636244032/fullscreen)のクラウド変数を使用しています。
